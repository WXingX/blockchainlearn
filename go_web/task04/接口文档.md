# 用户接口

## POST 用户注册

POST /user/register

> Body 请求参数

```json
{
  "username": "string",
  "password": "string",
  "email": "string"
}
```

### 请求参数

|名称|位置|类型|必选|说明|
|---|---|---|---|---|
|body|body|object| 否 |none|
|» username|body|string| 是 |用户名称|
|» password|body|string| 是 |密码|
|» email|body|string| 是 |邮箱|

> 返回示例

> 200 Response

```json
{}
```

### 返回结果

|状态码|状态码含义|说明|数据模型|
|---|---|---|---|
|200|[OK](https://tools.ietf.org/html/rfc7231#section-6.3.1)|none|Inline|
